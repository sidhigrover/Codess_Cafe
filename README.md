# Codess Cafe

## Install & Run the Project

# Install Bun
curl -fsSL https://bun.sh/install | bash
source ~/.zshrc

# Clone the repository
git clone <repo-url>
cd Codess_Cafe

# Install dependencies
bun install
## If Bun does not found, close VS Code and reopen it.

# Run the development server
bun run dev
